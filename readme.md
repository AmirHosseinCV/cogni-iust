# یادداشت‌های درس علوم شناختی محاسباتی

این مخزن یک مخزن موقت برای نگهداری یادداشت‌های جلسات درس «علوم شناختی محاسباتی» است. محتوای این یادداشت‌ها بر اساس جلسات درس زینب آقاهادی از دانشگاه علم و صنعت ایران تهیه شده است و ممکن است در ادامه ویرایش، تکمیل یا بازبینی شود.
